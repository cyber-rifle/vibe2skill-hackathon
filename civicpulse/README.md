# CivicPulse — Community Hero · Vibe2Ship 2026

AI-powered civic issue reporting for Indian communities. Upload a photo, and a 4-step Gemini agent chain analyzes, checks duplicates, assesses severity with live grounding, and drafts a routed municipal report.

## Stack
Next.js 16 · TypeScript · Tailwind v4 · Gemini 2.5 Flash · Google Search Grounding · React

## Features
- Gemini bounding box overlay — AI identifies the exact issue location in the photo frame
- 4-step sequential agent chain with a live ReasoningReveal animation
- Google Search grounding on severity assessment, with cited sources when available
- Human-in-the-loop confirmation panel before any report is submitted
- Live community map with color-coded severity markers
- Resolution time estimate sourced from real municipal data, where available
